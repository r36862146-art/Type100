# Coding Standards

Language

TypeScript only.

Components

- Small
- Reusable
- Single Responsibility

Functions

Pure whenever possible.

Naming

Components
PascalCase

Functions
camelCase

Hooks

useSomething()

Constants

UPPER_SNAKE_CASE

Files

kebab-case

Comments

Explain WHY.
Avoid obvious comments.

Imports

Absolute imports preferred.

Performance

Memoize expensive calculations.

Accessibility

Keyboard navigation required.

Testing

New logic should be testable.
